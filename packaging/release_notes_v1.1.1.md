## LiveBabel v1.1.1

本地优先的语音转写工具,三种模式一站搞定:

- 🎧 **实时字幕** — 抓电脑正在播放的声音,实时识别 + 翻译,桌面悬浮双语字幕。看直播 / 网课 / 外语视频。
- 🎬 **离线字幕** — 本地视频生成双语 SRT/ASS,可烧录进视频,支持多文件批量。
- 📝 **会议纪要** — 双流录制(我 / 远端),实时转录并声纹区分说话人,一键生成结构化纪要。

### ✨ 本次更新

- **修复**:部分笔记本(无可用 CUDA)上实时 / 会议模式因 GPU 初始化失败而闪退;现自动稳妥回退 CPU。
- **修复**:离线模式跑完后再开实时 / 会议,GPU 被占用无法加速的问题(离线转录改为独立进程,用完即彻底释放 GPU)。
- **新增**:实时模式可选「不翻译」,只显示识别原文(省 API、只想看原文时用)。
- **优化**:界面继续打磨(首页标题居中、字体更顺眼);项目内部结构重组,更清晰。

### 下载

本次发布 **CPU 版**(无需显卡,任何 Windows 电脑可用):

- `LiveBabel-CPU-v1.1.1-win64.zip`(约 550MB,**不含模型**)

### 首次使用 · 一步

双击 **`LiveBabel-CPU.exe`** 即可。首次运行会自动弹窗下载语音识别模型
(约 570MB,需联网,只需一次,已内置国内镜像加速),下完自动进入主页。

> 离线模式首次转写还会自动下载 Whisper 模型(约 1.6GB,由程序处理,仅一次)。
> 翻译功能需在主页「设置 Key」填入 DeepSeek API Key(不填也能识别 / 转录,只是不翻译)。

### 说明

- 模型不随包(太大),由程序首次运行时自动下载,之后开箱即用。
- 若自动下载失败,可点窗口里的「重试」(已下部分会续传),或运行随附的「下载模型.bat」手动下。
- 程序无数字签名,杀毒软件可能误报,选择信任 / 允许即可。
- 这是纯 CPU 版,识别速度比 GPU 慢一些,但兼容所有电脑。

### 系统要求

Windows 10 / 11 (64 位)。无需安装 Python / CUDA / ffmpeg。
